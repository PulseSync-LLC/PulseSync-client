## О проекте

**PulseSync** - это приложение для интеграции **Яндекс Музыки** с **Discord Rich Presence** и не только. Оно позволяет отображать информацию о текущем воспроизводимом треке в вашем профиле Discord. Также имеется поддержка аддонов, которые позволяют кастомизировать Яндекс Музыку.

> [!CAUTION]  
> ### Данная модификация НЕ даёт возможность пользоваться сервисом БЕЗ подписки Яндекс Плюс

## Установка

**Новую версию** можно скачать [на сайте](https://pulsesync.dev).
